# Public representative
This repository contains list of TON community public representatives as well as their PGP public keys.
Commits to this repository require approve from [all main developers](https://github.com/orgs/newton-blockchain/people).
## Public representative duty and privileges
Public representative has right to speak on behalf of community on issues related to the TON development.
In particular, but not limited to, community endorse public representatives to [acquire ownership](https://github.com/newton-blockchain/TIPs/issues/33) of the TON resources.

1. [@tolya-yanot](https://github.com/tolya-yanot) [PGP public key](#tolya-yanot-key);
